Руководство пользователя

Вход в систему

  1. Перейдите по URL-адресу админки.
  2. Введите ваше имя пользователя и пароль для входа в систему.

Основные элементы интерфейса

Панель администратора: Содержит элементы для управления предметами (шезлонги и кровати), кнопки для создания и удаления предметов.

Список бронирований: Здесь отображаются все бронирования. Вы можете фильтровать бронирования по дате прибытия или дате создания, а также загружать больше бронирований при необходимости.

Управление предметами

  1. Тип предмета: Выберите тип предмета (шезлонг или кровать) в выпадающем списке.
  2. Дата бронирования: Укажите дату, для которой хотите просмотреть предметы.
  3. Кнопки управления предметами:
    • Добавить предмет: Нажмите на кнопку "+" для добавления нового предмета.
    • Удалить предмет: Нажмите на кнопку "-" для удаления выбранного предмета.
    • Обновить предметы: Нажмите кнопку "Обновить" для обновления информации о предметах.
  4. Легенда цветов:
    • Желтый: Бронирования в процессе оплаты.
    • Красный: Бронирования успешно оплачены.

Работа с бронированиями

  1. Фильтрация бронирований: Укажите начальную и конечную даты для фильтрации бронирований. Выберите тип фильтрации (по дате прибытия или по дате создания).
  2. Просмотр бронирований: Бронирования будут отображаться в виде списка. Для каждого дня будут показаны бронирования, сделанные в этот день (или прибытие в этот день в зависимости от фильтрации).
  3. Кнопка "Сегодня": Нажмите на кнопку "Сегодня" для отображения всех бронирований на текущий день.
  4. Загрузка дополнительных бронирований: Если бронирований больше 50, будет отображена кнопка "Показать еще". Нажмите на эту кнопку для загрузки дополнительных бронирований.

Создание и изменение бронирований (В работе)

  1. Создание бронирования: Нажмите на кнопку "Создать бронирование" для открытия формы создания бронирования. Заполните все необходимые поля и нажмите кнопку "Сохранить" для завершения создания бронирования.
  2. Изменение бронирования: Найдите нужное бронирование в списке, внесите изменения и сохраните их.

Удаление бронирований

  1. Найдите бронирование, которое хотите удалить.
  2. Нажмите на кнопку удаления рядом с бронированием.
  3. Подтвердите удаление.

Логаут

Нажмите на кнопку "Выйти", расположенную в верхнем правом углу страницы, чтобы выйти из системы.
